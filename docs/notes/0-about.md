# About

- More on _functional programming_ rather than in "learning Scala"
- Book organization:
  1. Introduction to functional programming
  2. functional libraries and functional thought process
  3. higher abstractions for generalized libraries
  4. bridge gap between real-world I/O and functional programming

## Target Audience

- intermediate-level Java or C programming curious about functional programming.
- must be prepared to be a beginner again.
- does not require prior exprience with Scala.

## How to read this book

- Feel free to jump between and return to parts if necessary.
- Exercises are marked by whether they're essentiall or optional
